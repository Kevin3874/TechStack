## About The Project

Find and compare Tech across the internet's most popular websites.

* Currently supports: Amazon, BestBuy, Newegg
* Production server is NOT stable, will be updated 

In future implementations, users will be able to:
* Filter their searches
* Login and save items to their profile
* Share items with others

## Getting Started

To get it up and running (much more stable than what is deployed):

### Setting Up and Installation


**You can access the frontend using https://techstackshop.com, or you can run the server locally **

_Below is an example of how you can instruct your audience on installing and setting up your app. This template doesn't rely on any external dependencies or services._

1. Clone the repo
   ```sh
   git clone https://github.com/Kevin3874/TechStack.git
   ```
2. Install dependencies
   ```sh
    1. cd techstack-app
    2. npm install
    3. cd ../techstack-backend
    4. npm install
   ```
3. Run server
   ```sh
   node server.js
   ```
4. Run Client (Optional)
    ```sh
    either:
    - Go to https://techstackshop.com
    or
    cd ../techstack-app
    npm run dev
    ```

The server endpoint will be hosted at: http://localhost:4000

To access specific API endpoints, use the following format: http://localhost:4000/api/{endpoint}

You should be able to go on your local client, or to the hosted website, and search your item!
